# Jeu Donkey Kong (Python)

Petit jeu inspiré du classique Donkey Kong réalisé dans le cadre du cours d'informatique.

## Objectif
Le joueur doit éviter les obstacles et atteindre le sommet sans perdre toutes ses vies.

## Technologies utilisées
- Python
- pygame
  
## Installation
- git clone https://github.com/queenshkk/Jeu-Donkey-Kong.git
- cd Jeu-Donkey-Kong 
- pip install pygame 

## Lancer le jeu
python main.py
